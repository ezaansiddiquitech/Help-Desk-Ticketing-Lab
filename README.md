# IT Support Portfolio & Ticketing Lab

Welcome to my Help Desk portfolio. This repository serves as a practical demonstration of my technical troubleshooting abilities, system administration familiarity, and empathy-driven customer service communication. 

Instead of relying strictly on theoretical terms, this lab documents common real-world enterprise scenarios resolved using industry-standard technical workflows.

### 🛠️ Technical Skillset Demonstrated:
* **Networking Support:** DHCP troubleshooting (APIPA identification), DNS flushing, `ipconfig`, `ping`, and connectivity validation.
* **Operating Systems & Hardware:** Event Viewer logs analysis, Device Manager driver management, and Windows OS stability repair.
* **Identity & Access Management:** Microsoft Entra ID admin center workflow, user verification, and Multi-Factor Authentication (MFA) remediation.
* **Help Desk Operations:** Ticket documentation, root-cause analysis, and customer-facing soft skills.
